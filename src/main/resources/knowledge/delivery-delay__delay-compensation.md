# 배달 지연 보상 기준

배달이 예상 시간보다 지연된 경우 아래 기준으로 보상을 안내합니다.

## 지연 시간별 기준

| 지연 범위 | 보상 |
|---|---|
| 예상 시간 + 10분 이내 | 보상 대상 아님 (정상 범위) |
| 예상 시간 + 11~29분 | 다음 주문 사용 가능한 **1,000원 쿠폰** 자동 지급 |
| 예상 시간 + 30~59분 | **배달비 전액 환불** 또는 **3,000원 쿠폰** 중 선택 |
| 예상 시간 + 60분 이상 | **전액 환불** 검토 대상 (상담원 연결) |

## 보상 제외 사유

다음 사유로 인한 지연은 보상 대상이 아닙니다.

- **기상 특보** (태풍, 폭설, 폭우 경보): 배달 시간이 평소보다 길어질 수 있으며, 사전에 앱에서 안내됩니다.
- **고객 귀책**: 공동현관 비밀번호 미공유, 전화 불수신 등으로 인한 지연.
- **주소 오기재**: 고객이 입력한 주소가 실제 주소와 다른 경우.

## 자동 보상 처리

- 11분 이상 지연 시 별도 요청 없이 **자동으로 쿠폰이 지급**됩니다.
- 쿠폰 지급 알림은 앱 푸시 또는 주문 내역 상세에서 확인할 수 있습니다.

## 주의사항

- "비 오는 날"이라는 이유만으로는 보상이 어려울 수 있습니다. 기상 특보 여부를 함께 확인합니다.
- 보상 기준은 **배달 완료 시점**이 아닌 **예상 시간 기준**입니다.
