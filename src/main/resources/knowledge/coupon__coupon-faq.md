# 쿠폰 사용 FAQ

## 쿠폰 적용이 안 되는 경우

쿠폰이 정상적으로 적용되지 않는 가장 흔한 원인입니다.

- **최소 주문 금액 미달**: 쿠폰별로 정해진 최소 주문 금액을 충족해야 합니다.
- **적용 대상 카테고리 불일치**: 치킨 전용 쿠폰은 한식/분식 등에서는 사용 불가합니다.
- **중복 사용 제한**: 할인 쿠폰은 1회 1매만 사용 가능하며, 배달비 쿠폰과 함께는 사용 가능합니다.
- **사용 기한 만료**: 쿠폰 사용 기한이 지나면 자동으로 비활성화됩니다.

## 쿠폰 유효기간

- 발급 시점부터 쿠폰별로 표시된 **유효기간** 내에만 사용 가능합니다.
- 기한 연장은 원칙적으로 불가합니다.

## 이미 사용한 쿠폰을 복원할 수 있나요?

- 주문을 **취소한 경우**, 사용되었던 쿠폰은 자동으로 복원됩니다 (복원에는 최대 1시간 소요).
- 주문이 **정상 완료**된 경우, 쿠폰은 복원되지 않습니다.

## 쿠폰 중복 사용

- 할인 쿠폰 + 배달비 쿠폰 → 가능
- 할인 쿠폰 + 할인 쿠폰 → 불가
- 프로모션 쿠폰과 일반 쿠폰의 중복 여부는 쿠폰마다 다르며, 쿠폰 상세에 명시됩니다.
