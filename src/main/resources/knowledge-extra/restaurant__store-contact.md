# 가게(사장님) 연락처 안내 정책

## 사장님 개인 연락처는 제공하지 않습니다

- 가게 사장님의 **휴대폰 번호, 개인 이메일, 메신저 아이디** 등은 어떠한 경우에도 고객에게 제공하지 않습니다.
- 이는 자영업자의 개인정보 보호를 위한 원칙으로, 상담사나 AI 상담봇도 예외 없이 동일하게 적용됩니다.

## 가게에 직접 연락해야 할 때

- 앱의 가게 상세 페이지에 등록된 **가게 대표번호**로 연락해 주세요.
- 주문 내역 상단의 "가게에 전화" 버튼을 누르면 **안심번호**를 통해 연결됩니다.
- 안심번호는 주문 완료 후 **2시간 동안** 유효하며, 이후 자동 만료됩니다.

## 가게가 전화를 받지 않아요

- 영업 시간 내임에도 전화 연결이 어려운 경우 앱 내 "1:1 문의"로 접수해 주세요.
- 상담사가 가게에 직접 확인 후 결과를 안내합니다.

## 요청 주문 (알레르기, 특이사항 전달)

- "땅콩 알레르기", "간 조절 부탁" 등은 주문 시 **요청사항란**에 입력하여 가게에 전달됩니다.
- 주문 완료 후 변경이 필요한 경우 안심번호로 가게에 직접 연락해 주세요.
