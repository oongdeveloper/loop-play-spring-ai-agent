# 결제 오류 대응 가이드

결제가 비정상 종료되었거나 중복 결제된 경우 아래 절차를 따라 주세요.

## 결제 실패인데 카드에는 승인 내역이 있어요

- 앱에서 결제 실패 처리되었으나 카드사 문자로 승인 알림이 온 경우, 이는 **가승인** 상태입니다.
- 가승인 건은 **최대 5영업일 이내** 자동 취소 처리됩니다.
- 5영업일이 지나도 취소되지 않으면 고객센터 **1600-0987**로 문의해 주세요.

## 같은 주문에 두 번 결제됐어요

- 네트워크 지연으로 중복 결제가 발생한 경우, 시스템이 **자동 감지**하여 동일 금액 건 하나를 즉시 취소합니다.
- 자동 감지되지 않은 경우 앱 내 "1:1 문의"에서 주문번호와 카드 승인 번호를 전달해 주세요.

## 결제 중 앱이 종료됐어요

- 결제 진행 중 앱이 종료된 경우, 주문이 실제로 접수되었는지 "주문내역"에서 확인하세요.
- 주문내역에 없고 카드 승인만 된 경우는 가승인 건으로 자동 취소됩니다.
